# Derivatives of Hyperbolic Functions

| Function $f(x)$ | Derivative $f'(x)$ |
| -------- | ------- |
| $\sinh(x)$ | $\cosh(x)$ |
| $\cosh(x)$ | $\sinh(x)$ |
| $\tanh(x)$ | $\text{sech}^2(x)$ |
| $\text{sech}(x)$ | $-\text{sech}(x)\tanh(x)$ |
| $\text{csch}(x)$ | $-\text{csch}(x)\coth(x)$ |
| $\coth(x)$ | $-\text{csch}^2(x)$ |

| Function $f(x)$ | Derivative $f'(x)$ |
| -------- | ------- |
| $\sinh^{-1}(x)$ | $\frac{1}{\sqrt{x^2+1}}$ |
| $\cosh^{-1}(x)$ | $\frac{1}{\sqrt{x^2-1}}$ |
| $\tanh^{-1}(x)$ | $\frac{1}{1-x^2}$ |
| $\text{sech}^{-1}(x)$ | $-\frac{1}{x\sqrt{1-x^2}}$ |
| $\text{csch}^{-1}(x)$ | $-\frac{1}{\mid x\mid\sqrt{1+x^2}}$ |
| $\coth^{-1}(x)$ | $\frac{1}{1-x^2}$ |
